# Caves game
## Info

This is a game I've made during my spare time in 2020, and it was a learning experience for me.

## Goal

Escape the caves.

## Getting Started

To build the place from scratch, use:

```bash
rojo build -o main.rbxl
```

Next, Open the "main.rbxl" and play the game.
